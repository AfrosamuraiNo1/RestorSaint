## Installation

Установите зависимости.
bash
  python -m pip install -r requirements.txt
Сделать скрипт исполняемым на Linux.
bash
  chmod +x run.sh
Запустить скрипт на Linux.
bash
  ./run.sh
Перейменовать(.bat) и запустить скрипт на Windows.
bash
  run.bat
